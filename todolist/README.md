# <1> React 프로젝트 생성하기
### (1) npm install -g create-react-app
### (2) create-react-app project-이름 (원하는 project명 입력하기 React는 프로젝트 명에 대문자나 특수문자 기호 사용하면 안됨)
### (3) cd project-이름
### (4) npm start
